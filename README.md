# Kotha Family Invitation Website

Static GitHub Pages site for the House Warming and Satyanarayana Swamy Vratham invitation.

## Publish With GitHub Pages

1. Create a public GitHub repository.
2. Push these files to the repository's `main` branch.
3. In GitHub, open `Settings` -> `Pages`.
4. Set `Source` to `Deploy from a branch`.
5. Select `main` and `/root`, then save.
6. The live link will be:

```text
https://<your-github-username>.github.io/<repository-name>/
```

## Music

The public site includes `assets/telugu-festive-song.wav`.

Locally supplied movie-song files are intentionally ignored by git because public redistribution may require rights from the copyright owner.
